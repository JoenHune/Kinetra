### 如何撰写旅程日志

每篇日志应包含：

1. **背景** — 当时面临的问题或目标
2. **调研** — 参考了什么论文、库、方法
3. **决策** — 做了什么决定，为什么
4. **结果** — 实现了什么，跑通了哪些测试
5. **反思** — 学到了什么，下一步是什么
